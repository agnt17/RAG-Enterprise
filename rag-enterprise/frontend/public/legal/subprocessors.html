Subprocessors

Last updated: April 18, 2026

This page lists key third-party service providers that may process data on behalf of DocMind AI. It is maintained as a transparency reference and may be updated over time.

Provider Category Purpose Data Categories
Supabase Storage Private file storage and retrieval Uploaded documents, profile images, file metadata
Pinecone Vector Database Embedding storage and semantic retrieval Embeddings, vector metadata, document namespace IDs
Cohere AI/ML Embeddings and reranking Extracted text chunks, query text, ranking context
Groq AI/ML LLM answer generation User question and retrieved context excerpts
Razorpay Payments Payment processing and billing references Payment metadata, billing details, transaction IDs
Configured SMTP provider (for example, Gmail or SendGrid) Email Verification and transactional email delivery Email address, verification content
Render Infrastructure Hosting Backend hosting and runtime infrastructure Operational logs, service traffic metadata
Vercel Frontend Hosting Frontend deployment and delivery Web request metadata and delivery logs

Change Management

We may add, replace, or remove subprocessors as the product evolves. This page should be updated with material changes before or when new subprocessors are used in production.
